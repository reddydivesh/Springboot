Steps to execute.
1)Unizp the file.
2)In RestFileUploadController.java file Please set the path to any local drive in your computer, except "C: drive "
because "C: Drive" requires Administrative rights to transfer a file (by default it was set to E: drive in my program)
3)File size limit is set to 10MB.
4)Any document or file containing text is transferrable.
5)Run as Java Program and point the URL to URL:Portnumber/ (localhost:8080/) and it loads the upload.html.
6)Choose any file and click on "submit" . then check your local drive for the uploaded file.
